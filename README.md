# 🎲 Dice Pool Party — Feuille de Score

Application web progressive (PWA) pour suivre ses points au jeu **Dice Pool Party** de Gigamic.

## ✨ Fonctionnalités

- 🎲 **12 catégories de score** (6 couleurs + 6 spéciales)
- 📱 **Pavé numérique** au clic sur chaque case
- 🏆 **Compteur de victoires** avec enregistrement manuel
- ⭐ **Meilleur score** sauvegardé
- 💾 **Sauvegarde automatique** (localStorage)
- 🎊 **Confettis** en cas de victoire !
- 📲 **Installable** sur smartphone (PWA)

## 🚀 Déploiement sur GitHub Pages

1. Créer un repo GitHub (ex: `dice-pool-party`)
2. Uploader les fichiers : `index.html`, `manifest.json`, `favicon.svg`
3. Dans **Settings → Pages** → Branch: `main` → Save
4. L'app sera accessible sur `https://VOTRE_PSEUDO.github.io/dice-pool-party/`

## 📲 Installation sur smartphone

### Android
- Ouvrir l'URL dans Chrome
- Menu (⋮) → "Ajouter à l'écran d'accueil"

### iPhone
- Ouvrir dans Safari
- Bouton Partager → "Sur l'écran d'accueil"

## 🎮 Comment jouer

Chaque joueur a l'app sur son téléphone. La feuille de score a 4 colonnes (une par partie). Cliquez sur une case pour saisir vos points. Quand toutes les cases sont remplies, indiquez si vous avez gagné ou non !
